## 🔧 优化内容

### 1️⃣ 新增国内服务规则（5个）

| 规则 | 覆盖服务 | 作用 |
|------|---------|------|
| **Douyin** | 抖音、TikTok国内版 | 视频流、图片CDN直连 |
| **BiliBili** | 哔哩哔哩 | 视频流、弹幕、直播直连 |
| **WeChat** | 微信 | 语音、视频通话、朋友圈直连 |
| **ChinaMedia** | 小红书、快手等国内媒体 | 图片、视频CDN直连 |
| **Download** | BT/PT下载、迅雷 | 大文件下载直连，不消耗代理流量 |

### 2️⃣ 调整规则优先级

**优化前**（问题）：
```
1. 广告拦截
2. 局域网直连
3. 国内域名直连
4-16. AI、流媒体、社交等代理规则
17. 国内IP直连 ← 位置靠后，国内APP可能被误判
18. 兜底规则
```

**优化后**（正确）：
```
1. 广告拦截
2. 局域网直连
3-7. 国内服务直连 ← 新增，优先匹配
8. 国内域名直连
9-15. AI、流媒体、社交等代理规则
16. 国外代理
17. 国内IP直连 ← 双重保障
18. 兜底规则
```

## ⚠️ 注意事项

### 1. 规则更新
- 国内服务规则每24小时自动更新
- 如果新出现的国内APP加载慢，等待规则自动更新

### 2. 特殊情况
如果某个国内服务仍然加载慢，可能原因：
- **DNS污染**: 已配置国内DNS解决
- **ISP限速**: 运营商问题，配置无法解决
- **服务器问题**: 平台自身服务器问题

### 3. 港澳台内容
- B站港澳台内容仍然走直连
- 如需解锁港澳台内容，需要单独配置（当前未启用）

## 📝 配置说明

### 规则提供者来源
- `blackmatrix7/ios_rule_script` - 国内外服务规则（持续维护）
- `Loyalsoldier/clash-rules` - 基础分流规则

### 自动更新
- 规则每24小时自动更新一次
- 无需手动维护

### 流量消耗
- **国内流量**: 全部直连，不消耗代理流量
- **下载流量**: BT/PT等大文件直连，节省代理流量
- **国外流量**: 继续走代理，不受影响
